# ML Project 2 — Unsupervised Learning (Clustering) on E-Commerce Products

This repository contains a Jupyter Notebook for **Project 2** (Unsupervised Learning / Clustering).  
We use an **E-Commerce Services** dataset (Kaggle) and apply preprocessing, exploratory analysis, dimensionality reduction, and clustering to discover groups of similar products.

## Contents
- `Proj2_Andjela_Milica_final.ipynb` — final notebook (full analysis + results)

## Dataset
The dataset is selected from Kaggle under the **E-Commerce Services** tag.  
Each row represents one product and includes:
- numerical attributes (price/discount, ratings, sales, stock, delivery, weights, scores)
- categorical attributes (category, brand, city, color, size, returnability, payment modes)
